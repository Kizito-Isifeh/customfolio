# 📄 Customfolio – Product Requirements Document

**Tagline:** *Don’t Just Apply. Stand Out.*

---

## 🧠 Overview

Customfolio is a job application assistant designed for creatives and tech professionals. It helps users craft tailored portfolios for each job they apply to, track their applications, and stand out from the crowd.

---

## 🎯 Goals

- Empower job seekers to tailor their applications and portfolios  
- Simplify project and application tracking  
- Create a confidence-boosting platform for standout job applications  

---

## 👥 Target Users

- Designers, developers, PMs, marketers  
- Junior tech professionals and career switchers  
- Bootcamp graduates and portfolio-driven job seekers  

---

## 🧩 Core Features

### 1. User Accounts

- Sign up/login via email and Google  
- Personal dashboard with saved portfolios and application data  

### 2. Project Management

- Upload project data: title, description, stack, media, links  
- Organize by industry category (e.g. Fintech, Health, AI)  

### 3. Custom Portfolio Builder

- Title the Application  
- Select relevant projects per application  
- Customize layout, visuals, and content  
- Real-time preview of the portfolio  

#### Custom URLs

- Branded links (e.g. `customfolio.com/kizito/stripe`)  
- Option to download as PDF  
- View analytics: link clicks, views, source referrer  

#### Templates

- Layout options: grid, clean, bold, case study, etc.  
- Brand customization: font, color, logo  

### 4. Application Tracker

- Manage status: applied, interview, offer  
- Connect jobs to portfolios used  
- Email reminders and progress tracking  

### 5. Crafting Shots (Mini Case Studies)

**Shots** are bite-sized, focused texts/case studies designed to showcase relevant experience for specific job roles.

#### 🔹 Features

- Title, short context, goal, outcome  
- Attach visuals/link (Figma, screenshots, prototypes)  
- Align to job requirements (e.g., “Led design system overhaul for fintech app”)  
- We are calling GPT API here  

#### 🛠 How It Works

- Users create and save reusable “Shots”  
- During Custom portfolio creation, users select relevant shots  
- Shots appear in a dedicated section or embedded in case studies  

#### ✨ Purpose

- Helps users focus their pitch  
- Easier than writing full-length case studies every time  
- Adds evidence to claims like “I'm a great communicator” or “I’ve shipped products in health/AI/finance”

## 🔧 Tech Stack

### Frontend

- **Framework:** React.js, Next.js  
- **Styling:** TailwindCSS  
- **Hosting:** Vercel  

### Backend

- **Language:** Node.js + Express or Django  
- **Database:** PostgreSQL or MongoDB  
- **Auth:** Firebase Auth or Auth0  
- **Storage:** Firebase Storage or AWS S3  
- **Analytics:** Mixpanel, PostHog, or Google Analytics  

---

## 🛠️ MVP Scope

- User registration/login  
- Project upload + categorization  
- Basic Customfolio builder  
- Branded portfolio URLs  
- Starter templates
- Application tracker
- Shots Crafter

---

## ✅ Success Metrics

- % of users who build and send at least 1 Customfolio  
- Conversion from waitlist to active users  
- Monthly active users (MAU)  
- User satisfaction (CSAT/NPS)  

---

## 💡 Why It Works

- Personalized applications increase interview chances  
- There's no easy tool for building dynamic, targeted portfolios—yet  
- It fulfills an emotional need: regaining control and confidence during job hunts
