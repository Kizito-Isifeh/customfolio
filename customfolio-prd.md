
# 📄 Customfolio – Product Requirements Document

**Tagline:** _Don’t Just Apply. Stand Out._

---

## 🧠 Overview

Customfolio is a job application assistant designed for creatives and tech professionals. It helps users craft tailored portfolios for each job they apply to, track their applications, and stand out from the crowd.

---

## 🎯 Goals

- Empower job seekers to tailor their applications and portfolios
- Simplify project and application tracking
- Create a confidence-boosting platform for standout job applications

---

## 👥 Target Users

- Designers, developers, PMs, marketers
- Junior tech professionals and career switchers
- Bootcamp graduates and portfolio-driven job seekers

---

## 🧩 Core Features

### 1. User Accounts
- Sign up/login via email and Google
- Personal dashboard with saved portfolios and application data

### 2. Project Management
- Upload project data: title, description, stack, media, links
- Organize by industry category (e.g. Fintech, Health, AI)

### 3. Customfolio Builder
- Select relevant projects per application
- Customize layout, visuals, and content
- Real-time preview of the portfolio

### 4. Custom URLs
- Branded links (e.g. `customfolio.com/kizito/stripe`)
- View analytics: link clicks, views, source referrer

### 5. Templates
- Layout options: grid, clean, bold, case study, etc.
- Brand customization: font, color, logo

### 6. Integrations
- Apply via LinkedIn, Wellfound, job boards
- Resume and cover letter upload
- Optional video/voice intro with Loom or similar

### 7. Application Tracker
- Manage status: applied, interview, offer
- Connect jobs to portfolios used
- Email reminders and progress tracking

---

## 🔧 Tech Stack

### Frontend
- **Framework:** React.js, Next.js
- **Styling:** TailwindCSS
- **Hosting:** Vercel

### Backend
- **Language:** Node.js + Express or Django
- **Database:** PostgreSQL or MongoDB
- **Auth:** Firebase Auth or Auth0
- **Storage:** Firebase Storage or AWS S3
- **Analytics:** Mixpanel, PostHog, or Google Analytics
